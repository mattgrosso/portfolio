Game Table

Game Table uses the boardgamegeek.com API to help board game groups choose what game to play.

Given a BGG username the user's collection can be filtered down by player count, duration, rating and genre.

The app then helps the users choose one game to play using a number of possible methods.

If you don't have a BGG username you can use mine: 'mattgrosso'.
