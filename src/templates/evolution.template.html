Evolution

Evolution is an ongoing project that will almost certainly never 'end'.

Essentially it is a life simulator. With each iteration the simulation becomes more complex.

At this point the creatures move about eating each other and procreating until they either overwhelm the ecosystem or all die of old age.

This project is really just for fun but it's fun to learn new things so the next step is to implement paper.js for better handling of the animations.

Tech stack

  • jQuery
  • Canvas
  • Paper.js
